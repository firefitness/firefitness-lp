# FIREFITNESS Looker Studio ダッシュボード設計

## 概要
GA4データをLooker Studio（旧データポータル）で可視化し、マーケティング効果を一目で把握できるダッシュボードを構築します。

---

## ダッシュボード構成

### ページ1: 概要サマリー

| 指標 | 表示形式 |
|------|----------|
| 総セッション数 | スコアカード |
| 総ユーザー数 | スコアカード |
| 総LINE登録クリック数 | スコアカード |
| 総予約クリック数 | スコアカード |
| 平均滞在時間 | スコアカード |
| 直帰率 | スコアカード |

**グラフ**
- 日別セッション数（折れ線グラフ）
- 流入元別セッション（円グラフ）

---

### ページ2: 流入元分析

| セクション | 内容 |
|------------|------|
| 流入元別テーブル | utm_source × セッション × LINEクリック × 予約クリック |
| チャネル別グラフ | Organic / Paid / Direct / Referral |
| キャンペーン別 | utm_campaign × コンバージョン数 |

**フィルター**
- 期間選択
- デバイス（mobile/desktop/tablet）
- ページ（LP別）

---

### ページ3: コンバージョン分析

| 指標 | 説明 |
|------|------|
| LINE登録クリック率 | LINEクリック / セッション |
| 予約クリック率 | 予約クリック / セッション |
| 電話クリック数 | tel:リンククリック |

**ファネル表示**
```
セッション → スクロール75% → LINEクリック → LINE登録完了
```

---

### ページ4: ページ別分析

| ページ | 計測項目 |
|--------|----------|
| /index.html | PV, 滞在時間, スクロール深度 |
| /mamarestore | PV, LINEクリック率, 予約クリック率 |

## Looker Studio セットアップ手順

### 1. データソース接続

1. [Looker Studio](https://lookerstudio.google.com/) にアクセス
2. 「作成」→「データソース」
3. 「Google Analytics」を選択
4. GA4プロパティ「G-QN2VNZB80Z」を選択
5. 接続を承認

### 2. レポート作成

1. 「作成」→「レポート」
2. 上記データソースを選択
3. テンプレートまたは空白から開始

### 3. カスタムイベント指標の追加

以下のイベントをGA4から取得：
- `line_click` - LINEボタンクリック
- `reserve_click` - 予約ボタンクリック
- `phone_click` - 電話クリック
- `scroll_depth` - スクロール深度
- `time_on_page` - ページ滞在時間

### 4. ディメンション設定

| ディメンション | 用途 |
|----------------|------|
| sessionSource | utm_source |
| sessionMedium | utm_medium |
| sessionCampaign | utm_campaign |
| pagePath | ページ別分析 |
| deviceCategory | デバイス別 |

---

## 共有設定

1. ダッシュボード右上「共有」
2. 「リンクを取得」→「閲覧者」に設定
3. URLをブックマーク

---

## 更新頻度

- GA4データは約24〜48時間遅れで反映
- ダッシュボードは自動更新
- リアルタイムデータはGA4で直接確認
